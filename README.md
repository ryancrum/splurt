# splurt
The Useless Terminal Jpeg Viewer

splurt requires a 256-color compatible terminal emulator

## Usage

Just run `splurt path/to/image.jpg` and behold the splendor.

![Engage](doc/screen.jpg)

## Build

Splurt uses cmake.

First: `cmake .`
Then: `make`
